# Arpan Mukherjee - Portfolio

Developer × Marketer portfolio built with React, Vite, and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy to Vercel:
1. Push to GitHub
2. Import project in Vercel
3. Deploy

## Customize

Update social links in `src/components/Contact.jsx`
